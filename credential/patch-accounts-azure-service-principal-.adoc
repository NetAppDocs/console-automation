= Update service principal credentials

[.api-doc-operation .api-doc-operation-patch]#PATCH# [.api-doc-code-block]#`/accounts/{accountPublicId}/azure/service-principal/{credentialsId}`#



Updates existing credentials using the specified account public ID.

**Token usage:** NetApp Console user token.
