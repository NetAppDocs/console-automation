= Retrieve credentials using the ID

[.api-doc-operation .api-doc-operation-get]#GET# [.api-doc-code-block]#`/accounts/{accountPublicId}/aws/assume-role/{credentialsId}`#



Retrieves existing ARN credentials using the specified account public ID.

**Token usage:** Both a NetApp Console user token and service token. The service token needs to include the following scopes: credentials:internal. Pass the user token or service token in the Authorization request header.
