= Delete credentials

[.api-doc-operation .api-doc-operation-delete]#DELETE# [.api-doc-code-block]#`/accounts/{accountPublicId}/credentials/{credentialsId}`#



Deletes generic credentials.

**Token usage:** Both a NetApp Console user token and service token. The service token needs to include the following scopes: credentials:internal. Pass the user token or service token in the Authorization request header.


== Parameters

|===
|Name
|Type
|In
|Required

|accountPublicId
|string
|path
|True
a|

|credentialsId
|string
|path
|True
a|

|===

== Response

```
Status: 200, Successful deleted generic credentials return OK
```



== Error

```
Status: 404, Credentials not found
```
