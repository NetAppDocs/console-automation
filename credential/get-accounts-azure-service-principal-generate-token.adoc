= Generate token from ID

[.api-doc-operation .api-doc-operation-get]#GET# [.api-doc-code-block]#`/accounts/{accountPublicId}/azure/service-principal/{credentialsId}/generate-token`#



Generates an Azure token by using existing service principal credentials.

**Token usage:** Both a NetApp Console user token and service token. The service token needs to include the following scopes: credentials:internal. Pass the user token or service token in the Authorization request header.
