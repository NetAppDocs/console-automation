= Retrieve credentials by type

[.api-doc-operation .api-doc-operation-get]#GET# [.api-doc-code-block]#`/accounts/{accountPublicId}/credentials`#



Retrieves a list of generic credentials by credentials type with a specified account public ID.

**Token usage:** Both a NetApp Console user token and service token. The service token needs to include the following scopes: credentials:internal. Pass the user token or service token in the Authorization request header.
