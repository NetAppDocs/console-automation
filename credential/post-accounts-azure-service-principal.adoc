= Add service principal credentials

[.api-doc-operation .api-doc-operation-post]#POST# [.api-doc-code-block]#`/accounts/{accountPublicId}/azure/service-principal`#



Creates new ARN credentials for a specified account public ID.

**Token usage:** NetApp Console user token.


== Parameters

[cols=5*,options=header]
|===
|Name
|Type
|In
|Required
|Description

|accountPublicId
|string
|path
|True
a|

|isGov
|boolean
|query
|False
a|Indicates whether the service principal is located in the AWS GovCloud (US) location


|x-simulator
|string
|header
|False
a|

|===
