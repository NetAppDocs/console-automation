= Rename a workspace

[.api-doc-operation .api-doc-operation-patch]#PATCH# [.api-doc-code-block]#`/tenancy/account/{accountId}/workspace/{workspacePublicId}`#



Renames a workspace with a specified new name.
            +
*Token usage:* BlueXP user token.
