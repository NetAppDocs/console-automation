= Service account endpoints in the Accounts REST API



+++<fieldset>+++The service account endpoints represent a general BlueXP service. A service account can be created as an account administrator or delegated more limited authority over one or more workspaces.

Each service account contains the following information:

* Public identifier (such as do0uPCFi)
* Name
* Auth0 identifier (such as HEgEEJbfE0UfBzhmyprOjdZskl3H8KT3)

The following actions can be performed with the service account endpoints:

* Create, Read, Update, and Delete (CRUD) operations
* Generate the service account secret+++</fieldset>+++
