= Retrieve user role and permissions

[.api-doc-operation .api-doc-operation-get]#GET# [.api-doc-code-block]#`/tenancy/account/{accountId}/permissions-for-user`#



Retrieves the user role and permissions of the specified account.
            +
*Token usage:* BlueXP user token.

|Type
