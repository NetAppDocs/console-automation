= Move account data

[.api-doc-operation .api-doc-operation-post]#POST# [.api-doc-code-block]#`/tenancy/account/{srcAccountId}/dest/account/{dstAccountId}`#



Moves account data from the source account to the destination account.
            +
*Token usage:* BlueXP service token with the following scopes: tenancy:support.
