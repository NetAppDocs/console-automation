= Retrieve user account workspaces

[.api-doc-operation .api-doc-operation-get]#GET# [.api-doc-code-block]#`/tenancy/account/{accountId}/user/{userId}/workspaces`#



Retrieves specified user account workspaces.
            +
*Token usage:* BlueXP service token with the following scopes: tenancy:external.
