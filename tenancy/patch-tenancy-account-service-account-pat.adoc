= Update details

[.api-doc-operation .api-doc-operation-patch]#PATCH# [.api-doc-code-block]#`/tenancy/account/{accountId}/service-account/{serviceAccountId}`#



Updates the service account details of a specified account.
      +
*Token usage:* BlueXP user token.


== Parameters

[cols=5*,options=header]
|===

|Name
|Type
|In
|Required
|Description

|accountId
|string
|path
|True
a|

|serviceAccountId
|string
|path
|True
a|Service Account Client ID. Do not provide the Service Account Public ID.

* example: BjwDgGkg28cLKg70pLUwz9rY87Ff5xHK


|authorization
|string
|header
|True
a|Authorization JWT access token


|X-User-Token
|string
|header
|False
a|User JWT access token


|X-Service-Request-Id
|string
|header
|False
a|Request identifier

|===

== Request Body

                

[cols=4*,options=header]
|===
|Name
|Type
|Required
|Description

|serviceAccountName
|string
|False
a|Service account name


|rolePublicId
|string
|False
a|Service account role


|workspacePublicIds
|array[string]
|False
a|Service account workspaces


|===

  "workspacePublicIds": [
    "string"
  ]
}
