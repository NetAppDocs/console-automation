= Discover an external resource

[.api-doc-operation .api-doc-operation-post]#POST# [.api-doc-code-block]#`/tenancy/resource/discover`#



Discovers an external resource to be added to BlueXP.
            +
*Token usage:* BlueXP user token.


== Parameters

[cols=5*,options=header]
|===

|Name
|Type
|In
|Required
|Description

|authorization
|string
|header
|True
a|Authorization JWT access token


|X-Service-Request-Id
|string
|header
|False
a|Request identifier

|===
