= Delete a resource from an account

[.api-doc-operation .api-doc-operation-delete]#DELETE# [.api-doc-code-block]#`/tenancy/resource/{resourceIdentifier}`#



Deletes a resource from a specified account. accountId must be passed to the X-Tenancy-Account-Id request header.
            +
*Token usage:* BlueXP user token.

|string

|False

|string
|header
|False
a|
