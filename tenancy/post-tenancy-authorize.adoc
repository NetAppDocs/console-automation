= Authorize an external resource

[.api-doc-operation .api-doc-operation-post]#POST# [.api-doc-code-block]#`/tenancy/authorize`#



Authorizes an external resource to perform specific actions.
            +
*Token usage:* BlueXP user token.


== Parameters

[cols=5*,options=header]
|===

|Name
|Type
|In
|Required
|Description

|authorization
|string
|header
|True
a|Authorization JWT access token


|X-User-Token
|string
|header
|False
a|User JWT access token


|X-Service-Request-Id
|string
|header
|False
a|Request identifier

|===
