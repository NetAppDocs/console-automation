= Authorize an agent

[.api-doc-operation .api-doc-operation-post]#POST# [.api-doc-code-block]#`/tenancy/authorize-agent-in-account`#



Authorizes an agent in a specified account.
            +
*Token usage:* Both a BlueXP user token and service token. Pass the user token in the Authorization request header and the service token in the X-Agent-Token request header.
