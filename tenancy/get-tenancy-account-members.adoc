= Retrieve user and service account data

[.api-doc-operation .api-doc-operation-get]#GET# [.api-doc-code-block]#`/tenancy/account/{accountId}/members`#



Retrieves the user and service account data of the specified account.
            +
*Token usage:* BlueXP user token.

|Name
|Type

|string

|X-User-Token

|False
