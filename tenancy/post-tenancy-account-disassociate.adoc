= Dissociate a user

[.api-doc-operation .api-doc-operation-post]#POST# [.api-doc-code-block]#`/tenancy/account/{accountId}/disassociate`#



Disassociates the user from the specified account.
            +
*Token usage:* BlueXP user token.
