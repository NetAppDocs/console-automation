= Create a workspace

[.api-doc-operation .api-doc-operation-post]#POST# [.api-doc-code-block]#`/tenancy/account/{accountId}/workspace/{name}`#



Creates a workspace in a specified account.
            +
*Token usage:* BlueXP user token.

|Type
