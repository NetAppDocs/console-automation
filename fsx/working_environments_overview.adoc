= Working environments endpoints in the Amazon FSx for NetApp ONTAP management REST API



+++<fieldset>+++The working environment endpoints manage operations made on working environments in Cloud Manager.

You can perform the following actions with the working environment endpoints:

* Create an FSx for ONTAP working environment
* Retrieve FSx for ONTAP working environments by account
* Discover an FSx for ONTAP working environment
* Retrieve volume details for an FSx for ONTAP working environment
* Retrieve ONTAP credentials for an FSx for ONTAP working environment
* Create ONTAP credentials for an FSx for ONTAP working environment
* Recover FSx for ONTAP as a working environment
* Retrieve details for an FSx for ONTAP working environment
* Delete an FSx for ONTAP working environment
* Remove an FSx for ONTAP working environment from a workspace
* Update the throughput capacity of an FSx for ONTAP working environment+++</fieldset>+++
