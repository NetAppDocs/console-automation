= Retrieve supported regions

[.api-doc-operation .api-doc-operation-get]#GET# [.api-doc-code-block]#`/aws/supported-regions`#



Retrieves all supported FSx regions by using the provided credentials.

**Token usage:** Cloud Manager user token.
