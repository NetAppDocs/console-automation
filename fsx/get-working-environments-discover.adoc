= Discover working environments

[.api-doc-operation .api-doc-operation-get]#GET# [.api-doc-code-block]#`/working-environments/{accountId}/discover`#



Discovers all FSx ONTAP working environments of an AWS region by using specified credentials.
 'inDifferentAccount' field  in response indicates that a user is already assigned to a different Tenancy account.

**Token usage:** Cloud Manager user token.
