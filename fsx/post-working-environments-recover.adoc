= Recover working environments

[.api-doc-operation .api-doc-operation-post]#POST# [.api-doc-code-block]#`/working-environments/{accountId}/recover`#



Recovers an unassigned FSx working environment related to a Tenancy account by using the `fsx:assigned` parameter.

**Token usage:** Cloud Manager user token.
