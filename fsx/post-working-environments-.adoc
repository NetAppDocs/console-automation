= Create FSx for ONTAP working environments

[.api-doc-operation .api-doc-operation-post]#POST# [.api-doc-code-block]#`/working-environments/{accountId}`#



Creates an FSx ONTAP working environment on the required AWS account and region.

**Token usage:** Cloud Manager user token.
