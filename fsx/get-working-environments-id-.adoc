= Retrieve working environments by account

[.api-doc-operation .api-doc-operation-get]#GET# [.api-doc-code-block]#`/working-environments/{accountId}/{workingEnvironmentId}`#



Retrieves FSx working environments by specifying the workingEnvironmentId parameter.
   Pass provider-details:'true'  parameter to get provider data.
   Pass `volumes:'true'`  parameter to retrieve volumes.

**Token usage:** Cloud Manager user token.
