= Retrieve route tables

[.api-doc-operation .api-doc-operation-get]#GET# [.api-doc-code-block]#`/aws/route-tables`#



Retrieves all route tables for a specified region and VPC by using the credentials.

**Token usage:** Cloud Manager user token.
