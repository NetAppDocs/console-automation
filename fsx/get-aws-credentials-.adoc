= Retrieve AWS credentials

[.api-doc-operation .api-doc-operation-get]#GET# [.api-doc-code-block]#`/aws-credentials/{accountId}`#



Retrieves all the AWS credentials for a specific tenancy account.

**Token usage:** Cloud Manager user token.
