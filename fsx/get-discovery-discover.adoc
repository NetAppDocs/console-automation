= Discover FSx ONTAP systems

[.api-doc-operation .api-doc-operation-get]#GET# [.api-doc-code-block]#`/discovery/{accountId}/discover`#



Discovers all FSx ONTAP file systems. All the relevant AWS regions will be scanned during this operation.

**Token usage:** Cloud Manager user token.
