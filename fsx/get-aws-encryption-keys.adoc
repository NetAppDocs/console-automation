= Retrieve encryption keys

[.api-doc-operation .api-doc-operation-get]#GET# [.api-doc-code-block]#`/aws/encryption-keys`#



Retrieves all encryption keys for a specified region and VPC by using the credentials.

**Token usage:** Cloud Manager user token.
