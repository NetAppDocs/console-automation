= Delete an existing user

[.api-doc-operation .api-doc-operation-delete]#DELETE# [.api-doc-code-block]#`/users/{userId}`#



Deletes an existing user. It is not possible to delete a user that has created existing systems. It is not possible to delete the logged in user.
