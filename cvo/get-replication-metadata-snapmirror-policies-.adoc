= Retrieve SnapMirror policies

[.api-doc-operation .api-doc-operation-get]#GET# [.api-doc-code-block]#`/replication/metadata/snapmirror-policies/{sourceWorkingEnvironmentId}/{targetWorkingEnvironmentId}`#



Retrieves SnapMirror policies based on the source and destination Cloud Volumes ONTAP systems.
