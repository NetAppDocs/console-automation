= Delete an existing Cloud Volumes ONTAP system

[.api-doc-operation .api-doc-operation-delete]#DELETE# [.api-doc-code-block]#`/gcp/vsa/working-environments/{workingEnvironmentId}`#



Deletes an existing Cloud Volumes ONTAP system, including all cloud resources created for the system (unless the localDelete flag is set to true).

|Type

|Required

|localDelete
|boolean
|query

|boolean
|query
|False
a|
