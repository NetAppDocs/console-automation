= Generate a new key pair and self-signed certificate at the key store

[.api-doc-operation .api-doc-operation-post]#POST# [.api-doc-code-block]#`/occm/key-store/external-certificate`#



Generates a new key pair and self-signed certificate at the key store and returns a certificate signing request (CSR).


== Request Body

                   

[cols=4*,options=header]
|===
|Name
|Type
|Required
|Description

|occmCertificate
|string
|True
a|

|privateKey
|string
|False
a|

|algorithm
|string
|False
a|

|===


.Example request
[%collapsible%closed]
====
[source,json,subs=+macros]
{
  "occmCertificate": "string",
  "privateKey": "string",
  "algorithm": "string"
}
====
