= Retrieve an AWS account ID for credentials

[.api-doc-operation .api-doc-operation-post]#POST# [.api-doc-code-block]#`/accounts/aws/validate-creds`#



Retrieves an AWS cloud account ID for the specified credentials.



CAUTION: This operation is deprecated.

== Request Body

                   

[cols=4*,options=header]
|===
|Name
|Type
|Required
|Description

|accountName
|string
|False
a|

|providerKeys
|link:#AwsKeysRequest[AwsKeysRequest]
|True
a|

|subscriptionId
|string
|False
a|

|===


.Example request
[%collapsible%closed]
====
[source,json,subs=+macros]
{
  "accountName": "string",
  "providerKeys": {
    "awsAccessKeys": {
      "accessKey": "string",
      "secretKey": "string"
    },
    "assumeRoleArn": "string"
  },
  "subscriptionId": "string"
}
====

== Error

```
Status: Default, Default response
```
