= Retrieve all non-premises system actions for a tenant ID

[.api-doc-operation .api-doc-operation-get]#GET# [.api-doc-code-block]#`/working-environments/actionRequired/{tenantId}`#



Returns all non-premises system actions required for a specific tenant ID.


== Parameters

[cols=5*,options=header]
|===
|Name
|Type
|In
|Required
|Description

|tenantId
|string
|path
|True
a|

|===

|True
