= Initialize an existing SnapMirror relationship

[.api-doc-operation .api-doc-operation-post]#POST# [.api-doc-code-block]#`/replication/initialize/{destinationWorkingEnvironmentId}/{destinationSvmName}/{destinationVolumeName}`#



Initializes an existing SnapMirror relationship. This method can only be performed when the relationship has already been established and is healthy, idle, and uninitialized. For example, if the initial transfer were interrupted, it can be restarted with this request.
