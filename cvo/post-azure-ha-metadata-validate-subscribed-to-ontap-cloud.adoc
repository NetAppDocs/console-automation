= Validate the Azure marketplace subscription

[.api-doc-operation .api-doc-operation-post]#POST# [.api-doc-code-block]#`/azure/ha/metadata/validate-subscribed-to-ontap-cloud`#



Validates that the current user is subscribed to the Cloud Volumes ONTAP product in the Azure marketplace.

[cols=4*,options=header]
|===
|Name
|Type
|Required
|Description

|True
a|

|string

|cloudProviderAccount
|string
|False
a|

|===
