= Create a new replication for an FSx system

[.api-doc-operation .api-doc-operation-post]#POST# [.api-doc-code-block]#`/replication/fsx`#



Creates a new replication to an FSx system.


== Request Body

                   

[cols=4*,options=header]
|===
|Name
|Type
|Required
|Description

|replicationRequest
|link:#ReplicationToFsxRequest[ReplicationToFsxRequest]
|True
a|

|replicationVolume
|link:#FsxVolumeRequest[FsxVolumeRequest]
|True
a|

|===


.Example request
[%collapsible%closed]
====
[source,json,subs=+macros]
{
  "replicationRequest": {
    "sourceWorkingEnvironmentId": "string",
    "destinationFsxId": "string",
    "sourceInterclusterLifIps": [
      "string"
    ],
    "policyName": "string",
    "scheduleName": "string",
    "sourceIpSpace": "string"
  },
  "replicationVolume": {
    "sourceSvmName": "string",
    "sourceVolumeName": "string",
    "destinationVolumeName": "string",
    "destinationSvmName": "string",
    "tieringPolicy": "string"
  }
}
====

== Error

```
Status: Default, Default response
```
