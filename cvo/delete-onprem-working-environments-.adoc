= Delete an on-premises ONTAP system

[.api-doc-operation .api-doc-operation-delete]#DELETE# [.api-doc-code-block]#`/onprem/working-environments/{workingEnvironmentId}`#



Removes an on-premises ONTAP system from the NetApp Console. The Console will no longer manage the system, but ONTAP is not affected.
