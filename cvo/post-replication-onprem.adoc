= Create a new replication for an on-premises system

[.api-doc-operation .api-doc-operation-post]#POST# [.api-doc-code-block]#`/replication/onprem`#



Creates a new replication to an on-premises system.


== Request Body

                   

[cols=4*,options=header]
|===
|Name
|Type
|Required
|Description

|replicationRequest
|link:#ReplicationRequest[ReplicationRequest]
|True
a|

|replicationVolume
|link:#OnPremVolumeRequest[OnPremVolumeRequest]
|True
a|

|===


.Example request
[%collapsible%closed]
====
[source,json,subs=+macros]
{
  "replicationRequest": {
    "sourceWorkingEnvironmentId": "string",
    "destinationWorkingEnvironmentId": "string",
    "sourceInterclusterLifIps": [
      "string"
    ],
    "destinationInterclusterLifIps": [
      "string"
    ],
    "policyName": "string",
    "scheduleName": "string",
    "sourceIpSpace": "string",
    "destinationIpSpace": "string"
  },
  "replicationVolume": {
    "sourceSvmName": "string",
    "sourceVolumeName": "string",
    "destinationVolumeName": "string",
    "destinationAggregateName": "string",
    "destinationSvmName": "string",
    "destinationCapacityTier": "string",
    "tieringPolicy": "string"
  }
}
====

```
Status: Default, Default response
```
