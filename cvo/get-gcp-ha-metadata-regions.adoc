= Retrieve Google Cloud regions

[.api-doc-operation .api-doc-operation-get]#GET# [.api-doc-code-block]#`/gcp/ha/metadata/regions`#



Retrieves Google Cloud regions where a Cloud Volumes ONTAP system can be created.
