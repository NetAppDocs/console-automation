= Check if a system with the specified name exists

[.api-doc-operation .api-doc-operation-get]#GET# [.api-doc-code-block]#`/working-environments/exists/{workingEnvironmentName}`#



Checks if a system with the specified name already exists. Returns true if a system with that name already exists, false otherwise.
