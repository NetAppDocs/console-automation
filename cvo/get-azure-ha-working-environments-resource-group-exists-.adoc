= Check if a resource group with the specified name exists

[.api-doc-operation .api-doc-operation-get]#GET# [.api-doc-code-block]#`/azure/ha/working-environments/resource-group-exists/{resourceGroupName}`#



Checks if a resource group with the specified name already exists. Returns true if a resource group with that name already exists, false otherwise.
