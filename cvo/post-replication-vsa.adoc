= Create a new replication for a Cloud Volumes ONTAP system

[.api-doc-operation .api-doc-operation-post]#POST# [.api-doc-code-block]#`/replication/vsa`#

|Name
|Type
|Required
|Description

|True

|replicationVolume
|link:#VsaVolumeRequest[VsaVolumeRequest]
