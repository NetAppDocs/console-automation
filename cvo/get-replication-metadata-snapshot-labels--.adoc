= Return a list of snapshot labels

[.api-doc-operation .api-doc-operation-get]#GET# [.api-doc-code-block]#`/replication/metadata/snapshot-labels/{workingEnvironmentId}/{svmName}/{volumeName}`#



Returns a list of snapshot labels from the volume that match labels in the provided list.
