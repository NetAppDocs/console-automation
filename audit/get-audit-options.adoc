= Retrieve audit options

[.api-doc-operation .api-doc-operation-get]#GET# [.api-doc-code-block]#`/audit/{accountId}/options`#



Retrieves filter options by a specified account ID.

**Token usage:** NetApp Console user token.

|Type

|===
