= Delete the current account

[.api-doc-operation .api-doc-operation-delete]#DELETE# [.api-doc-code-block]#`/api/accounts`#



Deletes the current account. WARNING: This action will delete the current account from the Copy and Sync API.

[cols=5*,options=header]
|===
|Name
|Type
|In
|Required
|Description

|x-account-id
|string
|header
|True
a|NetApp tenancy account ID


|===

```
Status: 200, Default Response
```
