= Delete the reports by a specified report ID

[.api-doc-operation .api-doc-operation-delete]#DELETE# [.api-doc-code-block]#`/api/accounts/{accountId}/reports`#



Deletes the reports by a specified report ID.


== Parameters

[cols=5*,options=header]
|===
|Name
|Type
|In
|Required
|Description

|reports
|array[]
|query
|True
a|Reports IDs to delete


|accountId
|string
|path
|True
a|NetApp tenancy account ID


|===

== Response

```
Status: 200, Default Response
```
