= Retrieve the IBM bucket folders

[.api-doc-operation .api-doc-operation-get]#GET# [.api-doc-code-block]#`/api/data-brokers/{id}/list-ibm-bucket-folders`#



Retrieves the IBM bucket folders from a NetApp data broker.



CAUTION: This operation is deprecated.
