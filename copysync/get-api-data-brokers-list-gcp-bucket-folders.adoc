= Retrieve the Google Cloud bucket folders

[.api-doc-operation .api-doc-operation-get]#GET# [.api-doc-code-block]#`/api/data-brokers/{id}/list-gcp-bucket-folders`#



Retrieves the Google Cloud bucket folders from a NetApp data broker.



CAUTION: This operation is deprecated.

== Parameters

[cols=5*,options=header]
|===
|Name
|Type
|In
|Required
|Description

|bucket
|string
|query
|True
a|Bucket


|prefix
|string
|query
|False
a|Path inside the container, to list folders in


|id
|string
|path
|True
a|Data Broker ID


|x-account-id
|string
|header
|True
a|NetApp tenancy account ID


|===

== Response

```
Status: 202, Default Response
```
