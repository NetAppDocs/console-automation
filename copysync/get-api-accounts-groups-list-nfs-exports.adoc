= Retrieve the NFS server exports

[.api-doc-operation .api-doc-operation-get]#GET# [.api-doc-code-block]#`/api/accounts/{accountId}/groups/{id}/list-nfs-exports`#



Retrieves the NFS server exports from a data broker.


== Parameters

[cols=5*,options=header]
|===
|Name
|Type
|In
|Required
|Description

|host
|string
|query
|True
a|NFS server hostname or IP


|isEFS
|boolean
|query
|False
a|Indicates if the is target EFS


|accountId
|string
|path
|True
a|NetApp tenancy account ID


|id
|string
|path
|True
a|Data Brokers Group ID


|===

== Response

```
Status: 202, Default Response
```
