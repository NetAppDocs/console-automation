= Reports endpoints in the NetApp Copy and Sync REST API



The report endpoints enable you to view reports of NetApp Console account such as the number of directories and files in an account, the distribution of file size, the depth and breadth of the directories, and more. You can also create or delete a report by using the report endpoints.
The following actions can be performed with the report endpoints:

* Retrieve reports (including statistics of an account)
* Create a report
* Delete a report
* Abort a report
