= Retrieve the S3 bucket folders

[.api-doc-operation .api-doc-operation-get]#GET# [.api-doc-code-block]#`/api/accounts/{accountId}/groups/{id}/list-s3-bucket-folders`#



Retrieves the S3 bucket folders from a NetApp data broker.


== Parameters

[cols=5*,options=header]
|===
|Name
|Type
|In
|Required
|Description

|bucket
|string
|query
|True
a|Bucket


|prefix
|string
|query
|False
a|Path inside the container, to list folders in


|accountId
|string
|path
|True
a|NetApp tenancy account ID


|id
|string
|path
|True
a|Data Brokers Group ID


|===

== Response

```
Status: 202, Default Response
```
