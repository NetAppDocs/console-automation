= Modify the source or target credentials

[.api-doc-operation .api-doc-operation-put]#PUT# [.api-doc-code-block]#`/api/accounts/{accountId}/relationships/{id}/credentials`#



Modifies the source or target credentials.


== Parameters

[cols=5*,options=header]
|===
|Name
|Type
|In
|Required
|Description

|id
|string
|path
|True
a|Relationship ID


|accountId
|string
|path
|True
a|NetApp tenancy account ID


|===

== Request Body

                   

[cols=4*,options=header]
|===
|Name
|Type
|Required
|Description

|source
|link:#source[source]
|False
a|

|target
|link:#target[target]
|False
a|

|===
