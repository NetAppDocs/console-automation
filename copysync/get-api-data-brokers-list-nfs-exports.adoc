= Retrieve the NFS server folders

[.api-doc-operation .api-doc-operation-get]#GET# [.api-doc-code-block]#`/api/data-brokers/{id}/list-nfs-exports`#



Retrieves the NFS server folders from a data broker.



CAUTION: This operation is deprecated.

== Parameters

[cols=5*,options=header]
|===
|Name
|Type
|In
|Required
|Description

|host
|string
|query
|True
a|NFS server hostname or IP


|isEFS
|boolean
|query
|False
a|Indicates if the is target EFS


|id
|string
|path
|True
a|Data Broker ID


|x-account-id
|string
|header
|True
a|NetApp tenancy account ID


|===

== Response

```
Status: 202, Default Response
```
