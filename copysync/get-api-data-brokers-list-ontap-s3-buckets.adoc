= Retrieve the ONTAP S3 buckets

[.api-doc-operation .api-doc-operation-get]#GET# [.api-doc-code-block]#`/api/data-brokers/{id}/list-ontap-s3-buckets`#



Retrieves the ONTAP S3 buckets accessible from a NetApp data broker.



CAUTION: This operation is deprecated.

== Parameters

[cols=5*,options=header]
|===
|Name
|Type
|In
|Required
|Description

|host
|string
|query
|True
a|ONTAP S3 hostname or IP


|port
|number
|query
|False
a|Server port


|id
|string
|path
|True
a|Data Broker ID


|x-account-id
|string
|header
|True
a|NetApp tenancy account ID


|access-key
|string
|header
|True
a|StorageGRID server access-key


|secret-key
|string
|header
|True
a|StorageGRID server secret-key


|===

== Response

```
Status: 202, Default Response
```
