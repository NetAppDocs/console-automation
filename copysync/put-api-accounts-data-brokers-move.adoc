= Move the data broker between groups

[.api-doc-operation .api-doc-operation-put]#PUT# [.api-doc-code-block]#`/api/accounts/{accountId}/data-brokers/{id}/move`#



Moves the data broker between groups.


== Parameters

[cols=5*,options=header]
|===
|Name
|Type
|In
|Required
|Description

|accountId
|string
|path
|True
a|NetApp tenancy account ID


|id
|string
|path
|True
a|Data Broker ID


|===

== Request Body

                   

[cols=4*,options=header]
|===
|Name
|Type
|Required
|Description

|targetGroupId
|string
|False
a|Target Group ID


|targetGroupName
|string
|False
a|Target Group Name - use in case you want to move to a new group


|===


.Example request
[%collapsible%closed]
====
[source,json,subs=+macros]
{
  "targetGroupId": "string",
  "targetGroupName": "string"
}
====

== Response

```
Status: 200, Default Response
```
