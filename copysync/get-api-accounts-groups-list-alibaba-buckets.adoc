= Retrieve the Alibaba buckets

[.api-doc-operation .api-doc-operation-get]#GET# [.api-doc-code-block]#`/api/accounts/{accountId}/groups/{id}/list-alibaba-buckets`#



Retrieves the Alibaba buckets accessible from a data broker.


== Parameters

[cols=5*,options=header]
|===
|Name
|Type
|In
|Required
|Description

|accountId
|string
|path
|True
a|NetApp tenancy account ID


|id
|string
|path
|True
a|Data Brokers Group ID


|access-key
|string
|header
|True
a|StorageGRID server access-key


|secret-key
|string
|header
|True
a|StorageGRID server secret-key


|===

== Response

```
Status: 202, Default Response
```
