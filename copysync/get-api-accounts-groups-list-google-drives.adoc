= Retrieve the Google Drives

[.api-doc-operation .api-doc-operation-get]#GET# [.api-doc-code-block]#`/api/accounts/{accountId}/groups/{id}/list-google-drives`#



Retrieves the Google drives objects.


== Parameters

[cols=5*,options=header]
|===
|Name
|Type
|In
|Required
|Description

|accountId
|string
|path
|True
a|NetApp tenancy account ID


|id
|string
|path
|True
a|Data Brokers Group ID


|private-key
|string
|header
|True
a|Google Drive service account private key


|client-email
|string
|header
|True
a|Google Drive service account client email


|subject
|string
|header
|True
a|Google Drive user email


|===

== Response

```
Status: 202, Default Response
```
