= Retrieve the Google Drive folders

[.api-doc-operation .api-doc-operation-get]#GET# [.api-doc-code-block]#`/api/accounts/{accountId}/groups/{id}/list-google-drive-folders`#



Retrieves the Google Drive folders.


== Parameters

[cols=5*,options=header]
|===
|Name
|Type
|In
|Required
|Description

|driveId
|string
|query
|True
a|Google Drive ID


|folderId
|string
|query
|False
a|Folder ID


|folderName
|string
|query
|False
a|Path inside the container, to list folders in


|accountId
|string
|path
|True
a|NetApp tenancy account ID


|id
|string
|path
|True
a|Data Brokers Group ID


|private-key
|string
|header
|True
a|Google Drive service account private key


|client-email
|string
|header
|True
a|Google Drive service account client email


|subject
|string
|header
|True
a|Google Drive user email


|===

== Response

```
Status: 202, Default Response
```
