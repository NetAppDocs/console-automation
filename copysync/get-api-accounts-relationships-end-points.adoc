= Retrieve the relationship endpoints

[.api-doc-operation .api-doc-operation-get]#GET# [.api-doc-code-block]#`/api/accounts/{accountId}/relationships/end-points`#



Retrieves the relationship endpoints.


== Parameters

[cols=5*,options=header]
|===
|Name
|Type
|In
|Required
|Description

|accountId
|string
|path
|True
a|NetApp tenancy account ID


|===

== Response

```
Status: 200, Default Response
```


[cols=4*,options=header]
|===
|Name
|Type
|Required
|Description

|relationships
|array[link:#relationships[relationships]]
|False
a|

|===

[%collapsible%closed]

[source,json,subs=+macros]
{
  "relationships": [
    {
      "source": {
        "id": "string",

          "provider": "string"
        }
      },
      "target": {
        "id": "string",
        "metadata": {
          "bucket": "string",
          "host": "string",

          "provider": "string"
        }
      }
    }
  ]
}
