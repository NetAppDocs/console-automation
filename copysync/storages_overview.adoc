= Storages endpoints in the NetApp Copy and Sync REST API



The storages endpoints provide information about the different source and target locations.
This API category displays only the endpoints that are part of a sync relationship in an account.

You can retrieve the following information by using the storages API endpoints:

* NFS hosts IP addresses
* SMB hosts IP addresses
* Azure storage accounts
* SFTP hosts IP addresses
* IBM hosts IP addresses
* StorageGRID hosts IP addresses
* Azure NetApp Files IP addresses
* Cloud Volumes ONTAP, On-premises, FSx Data LIF IP addresses
* ONTAP S3 endpoint IP addresses
* Amazon Elastic File System IP addresses
* Box app names
* Google drive subjects
