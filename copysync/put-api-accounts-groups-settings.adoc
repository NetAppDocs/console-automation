= Modify the group settings

[.api-doc-operation .api-doc-operation-put]#PUT# [.api-doc-code-block]#`/api/accounts/{accountId}/groups/{id}/settings`#



Modifies the data broker group settings.


== Parameters

[cols=5*,options=header]
|===
|Name
|Type
|In
|Required
|Description

|accountId
|string
|path
|True
a|NetApp tenancy account ID


|id
|string
|path
|True
a|Data Brokers Group ID


|===

== Request Body

                   

[cols=4*,options=header]
|===
|Name
|Type
|Required
|Description

|name
|string
|False
a|Group Name


|scanner
|link:#scanner[scanner]
|False
a|

|transferrer
|link:#transferrer[transferrer]
|False
a|

|===


.Example request
[%collapsible%closed]
====
[source,json,subs=+macros]
{
  "name": "string"
}
====

```
Status: 200, Default Response
```
