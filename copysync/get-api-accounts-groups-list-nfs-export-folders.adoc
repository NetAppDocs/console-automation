= Retrieve the NFS server folders

[.api-doc-operation .api-doc-operation-get]#GET# [.api-doc-code-block]#`/api/accounts/{accountId}/groups/{id}/list-nfs-export-folders`#



Retrieves the NFS server folders from a data broker.


== Parameters

[cols=5*,options=header]
|===
|Name
|Type
|In
|Required
|Description

|host
|string
|query
|True
a|NFS server hostname or IP


|exportName
|string
|query
|True
a|NFS server export


|isEFS
|boolean
|query
|False
a|Indicates if the is target EFS


|version
|string
|query
|False
a|NFS version supported versions 3, 4.0, 4.1, 4.2


|accessPoint
|string
|query
|False
a|EFS access point


|path
|string
|query
|False
a|NFS Export path


|accountId
|string
|path
|True
a|NetApp tenancy account ID


|id
|string
|path
|True
a|Data Brokers Group ID


|===

== Response

```
Status: 202, Default Response
```
