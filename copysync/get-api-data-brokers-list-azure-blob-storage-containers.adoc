= Retrieve the Azure blob storage containers

[.api-doc-operation .api-doc-operation-get]#GET# [.api-doc-code-block]#`/api/data-brokers/{id}/list-azure-blob-storage-containers`#



Retrieves the Azure blob storage containers.



CAUTION: This operation is deprecated.

|===
|Name
|Type
|In
|Required
|Description

|storageAccountName
|string

|True

|id
|string
|path
|True

|x-account-id
|string
|header
|True
a|NetApp tenancy account ID


|storage-account-connection-string
|string
|header
|True
a|Azure storage account connection-string


|===
