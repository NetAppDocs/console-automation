= Retrieve the SFTP server folders

[.api-doc-operation .api-doc-operation-get]#GET# [.api-doc-code-block]#`/api/accounts/{accountId}/groups/{id}/list-sftp-directories`#



Retrieves the SFTP server folders from a NetApp data broker.


== Parameters

[cols=5*,options=header]
|===
|Name
|Type
|In
|Required
|Description

|host
|string
|query
|True
a|SFTP server hostname or IP


|port
|number
|query
|True
a|SFTP server port


|rootDirectory
|string
|query
|True
a|SFTP server root directory path


|path
|string
|query
|False
a|SFTP server root directory path


|accountId
|string
|path
|True
a|NetApp tenancy account ID


|id
|string
|path
|True
a|Data Brokers Group ID


|username
|string
|header
|True
a|SFTP server username


|password
|string
|header
|True
a|SFTP server password


|===

== Response

```
Status: 202, Default Response
```
