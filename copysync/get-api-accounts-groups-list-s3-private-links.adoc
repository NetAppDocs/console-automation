= Retrieve the S3 private links

[.api-doc-operation .api-doc-operation-get]#GET# [.api-doc-code-block]#`/api/accounts/{accountId}/groups/{id}/list-s3-private-links`#



Retrieves the S3 private links.


== Parameters

[cols=5*,options=header]
|===
|Name
|Type
|In
|Required
|Description

|region
|string
|query
|True
a|AWS region


|accountId
|string
|path
|True
a|NetApp tenancy account ID


|id
|string
|path
|True
a|Data Brokers Group ID


|x-bucket-name
|string
|header
|False
a|S3 bucket- for UI use


|===

```
Status: 202, Default Response
```
