= Add a license

[.api-doc-operation .api-doc-operation-put]#PUT# [.api-doc-code-block]#`/api/accounts/{accountId}/licenses-v2`#



Adds a license.


== Parameters

[cols=5*,options=header]
|===
|Name
|Type
|In
|Required
|Description

|accountId
|string
|path
|True
a|NetApp tenancy account ID


|===

== Request Body

                   

[cols=4*,options=header]
|===
|Name
|Type
|Required
|Description

|serialNumber
|string
|True
a|Serial number for license


|nssKeyId
|string
|True
a|NSS Key ID


|===


.Example request
[%collapsible%closed]
====
[source,json,subs=+macros]
{
  "serialNumber": "string",
  "nssKeyId": "string"
}
====

Status: 200, Default Response
