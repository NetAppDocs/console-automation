= Retrieve the IBM buckets

[.api-doc-operation .api-doc-operation-get]#GET# [.api-doc-code-block]#`/api/accounts/{accountId}/groups/{id}/list-ibm-buckets`#



Retrieves the IBM buckets accessible from a NetApp data broker.


== Parameters

[cols=5*,options=header]
|===
|Name
|Type
|In
|Required
|Description

|host
|string
|query
|True
a|Server hostname or IP


|port
|number
|query
|True
a|Server port


|accountId
|string
|path
|True
a|NetApp tenancy account ID


|id
|string
|path
|True
a|Data Brokers Group ID


|access-key
|string
|header
|True
a|StorageGRID server access-key


|secret-key
|string
|header
|True
a|StorageGRID server secret-key


|===

== Response

```
Status: 202, Default Response
```
