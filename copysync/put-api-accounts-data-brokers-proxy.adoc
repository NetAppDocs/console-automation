= Update the proxy configuration for a data broker

[.api-doc-operation .api-doc-operation-put]#PUT# [.api-doc-code-block]#`/api/accounts/{accountId}/data-brokers/{id}/proxy`#



Updates the proxy configuration for a data broker.


== Parameters

[cols=5*,options=header]
|===
|Name
|Type
|In
|Required
|Description

|accountId
|string
|path
|True
a|NetApp tenancy account ID


|id
|string
|path
|True
a|Data Broker ID


|===

== Request Body

                   

[cols=4*,options=header]
|===
|Name
|Type
|Required
|Description

|host
|string
|True
a|Proxy host


|port
|string
|True
a|Proxy port


|userName
|string
|False
a|Proxy userName


|password
|string
|False
a|Proxy password


|===


.Example request
[%collapsible%closed]
====
[source,json,subs=+macros]
{
  "host": "string",
  "port": "string",
  "userName": "string",
  "password": "string"
}
====

== Response

```
Status: 202, Default Response
```
