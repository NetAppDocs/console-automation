= Create a new report

[.api-doc-operation .api-doc-operation-post]#POST# [.api-doc-code-block]#`/api/accounts/{accountId}/reports`#



Creates a new report.


== Parameters

[cols=5*,options=header]
|===
|Name
|Type
|In
|Required
|Description

|accountId
|string
|path
|True
a|NetApp tenancy account ID


|===
