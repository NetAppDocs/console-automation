= Retrieve the Google Cloud buckets

[.api-doc-operation .api-doc-operation-get]#GET# [.api-doc-code-block]#`/api/accounts/{accountId}/groups/{id}/list-gcp-buckets`#



Retrieves the Google Cloud buckets accessible from a NetApp data broker.


== Parameters

[cols=5*,options=header]
|===
|Name
|Type
|In
|Required
|Description

|accountId
|string
|path
|True
a|NetApp tenancy account ID


|id
|string
|path
|True
a|Data Brokers Group ID


|===

== Response

```
Status: 202, Default Response
```
