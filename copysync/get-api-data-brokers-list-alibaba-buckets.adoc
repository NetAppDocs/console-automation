= Retrieve the Alibaba buckets

[.api-doc-operation .api-doc-operation-get]#GET# [.api-doc-code-block]#`/api/data-brokers/{id}/list-alibaba-buckets`#



Retrieves the Alibaba buckets accessible from a data broker.



CAUTION: This operation is deprecated.

== Parameters

[cols=5*,options=header]
|===
|Name
|Type
|In
|Required
|Description

|id
|string
|path
|True
a|Data Broker ID


|x-account-id
|string
|header
|True
a|NetApp tenancy account ID


|access-key
|string
|header
|True
a|StorageGRID server access-key


|secret-key
|string
|header
|True
a|StorageGRID server secret-key


|===
