= Retrieve the AWS KMS keys

[.api-doc-operation .api-doc-operation-get]#GET# [.api-doc-code-block]#`/api/accounts/{accountId}/groups/{id}/list-aws-kms-keys`#



Retrieves the AWS KMS keys from a NetApp data broker.


== Parameters

[cols=5*,options=header]
|===
|Name
|Type
|In
|Required
|Description

|bucket
|string
|query
|True
a|Bucket


|accountId
|string
|path
|True
a|NetApp tenancy account ID


|id
|string
|path
|True
a|Data Brokers Group ID


|===

== Response

```
Status: 202, Default Response
```
