= Add NetApp support site credentials

[.api-doc-operation .api-doc-operation-put]#PUT# [.api-doc-code-block]#`/api/v2/accounts/{accountId}/nss-credentials`#



Adds the NetApp Support Site credentials.



CAUTION: This operation is deprecated.

== Parameters

[cols=5*,options=header]
|===
|Name
|Type
|In
|Required
|Description

|accountId
|string
|path
|True
a|NetApp tenancy account ID


|===

== Request Body

                   

[cols=4*,options=header]
|===
|Name
|Type
|Required
|Description

|nssUsername
|string
|True
a|NSS user


|nssPassword
|string
|True
a|NSS password


|===


.Example request
[%collapsible%closed]
====

{
  "nssUsername": "string",
  "nssPassword": "string"
}
====
