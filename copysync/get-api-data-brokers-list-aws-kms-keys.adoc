= Retrieve the AWS KMS keys

[.api-doc-operation .api-doc-operation-get]#GET# [.api-doc-code-block]#`/api/data-brokers/{id}/list-aws-kms-keys`#



Retrieves the AWS KMS keys from a NetApp data broker.



CAUTION: This operation is deprecated.

== Parameters

[cols=5*,options=header]
|===
|Name
|Type
|In
|Required
|Description

|bucket
|string
|query
|True
a|AWS Bucket


|id
|string
|path
|True
a|Data Broker ID


|x-account-id
|string
|header
|True
a|NetApp tenancy account ID


|===

== Response

```
Status: 202, Default Response
```
