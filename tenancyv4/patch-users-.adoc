= Update a user partially

[.api-doc-operation .api-doc-operation-patch]#PATCH# [.api-doc-code-block]#`/users/{user_id}`#



Updates the specified user with partial information.


== Parameters

[cols=5*,options=header]
|===
|Name
|Type
|In
|Required
|Description

|user_id
|string
|path
|True
a|ID of the user.

* example: 666a3f38-d4fa-5b62-a391-a69029758d32


|===
