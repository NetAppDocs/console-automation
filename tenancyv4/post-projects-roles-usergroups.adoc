= Create a new user group in a project

[.api-doc-operation .api-doc-operation-post]#POST# [.api-doc-code-block]#`/projects/{project_id}/roles/{role_id}/usergroups`#



Creates a new user group.


== Request Body

Indicates the desired values for the user group to be created.
                   

[cols=4*,options=header]
|===
|Name
|Type
|Required
|Description

|description
|string
|False
a|JSON string containing the description of the user group. Defined values are: * Minimum length: 1 * Maximum length: 254 * This string is internally validated to prevent XSS, Unicode, directory traversal, and SQL-injection attacks.


|id
|string
|False
a|Globally unique identifier of the resource.
Defined values are:

* Conforms to the UUIDv4 Schema


|isSystem
|string
|False
a|Specifies if an object is system generated.
Defined values are:

* "true"


|metadata
|link:#type_bxp_metadata[type_bxp_metadata]
|False
a|Metadata associated with the resource.


|name
|string
|True
a|JSON string containing a name for the user group.
Defined values are:

* Minimum length: 1
* Maximum length: 300
* This string is internally validated to prevent XSS, Unicode, directory traversal, and SQL-injection attacks.


|ownerOrganizationId
|string
|True
a|JSON string containing the ID of the organization.


|tags
|array[link:#tags[tags]]
|False
a|JSON array of tags (name/value pairs) representing metadata for the user group.


|type
|string
|True
a|Media type of the resource.
Defined values are:

* "application/vnd.netapp.bxp.usergroup"


|usergroupType
|string
|False
a|Specifies type of userGroup.
Defined values are:

* SYSTEM
* FEDERATED
* USERGROUP
Defaults to SYSTEM


|version
|string
|True
a|Version of the resource.
Defined values are:

* "1.0"


|===


.Example request
[%collapsible%closed]
====
[source,json,subs=+macros]
{
  "description": "This tag represents NetApp Sales Territory",
  "name": "fed-pepsico-com-samlp_admin",
  "ownerOrganizationId": "6544af09-8036-51e4-b579-bd30cb07b269",
  "type": "application/vnd.netapp.bxp.usergroup",
  "usergroupType": "USERGROUP",
  "version": "1.0"
}
====
