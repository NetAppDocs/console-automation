= Retrieve a role

[.api-doc-operation .api-doc-operation-get]#GET# [.api-doc-code-block]#`/roles/{role_id}`#



Retrieves the specified role.


== Parameters

[cols=5*,options=header]
|===
|Name
|Type
|In
|Required
|Description

|role_id
|string
|path
|True
a|ID of the role.

* example: 666a3f38-d4fa-5b62-a391-a69029758d32


|===

== Response

```
Status: 200, Returns the stored role in the JSON response body.
```


[cols=4*,options=header]
|===
|Name
|Type
|Required
|Description

|category
|string
|False
a|Specifies the classification of the role.

* enum: ["IAM", "ONTAP-MEDIATOR", "SNAPCENTER"]
* example: IAM


|description
|string
|False
a|JSON string containing the description of the role. Defined values are: * Minimum length: 1 * Maximum length: 254 * This string is internally validated to prevent XSS, Unicode, directory traversal, and SQL-injection attacks.


|id
|string
|True
a|Globally unique identifier of the resource.
Defined values are:

* Conforms to the UUIDv4 Schema


|isAgentRole
|string
|False
a|Specifies if the role can be used for a Agent.
Defined values are:

* "true"


|isServiceAccountRole
|string
|False
a|Specifies if the role can be used for a Service Account.
Defined values are:

* "true"


|isSystem
|string
|False
a|Specifies if an object is system generated.
Defined values are:

* "true"


|isUserRole
|string
|False
a|Specifies if the role can be used for a User.
Defined values are:

* "true"


|links
|array[link:#type_bxp_link[type_bxp_link]]
|False
a|

|metadata
|link:#type_bxp_metadata[type_bxp_metadata]
|False
a|Metadata associated with the resource.


|name
|string
|True
a|JSON string containing a name for the role.
Defined values are:

* Minimum length: 1
* Maximum length: 63
* This string is internally validated to prevent XSS, Unicode, directory traversal, and SQL-injection attacks.


|roleInternalName
|string
|False
a|JSON string containing a internal name for the role used by UI to validate role name changes. Value will be same as internalName in private field
Defined values are:

* Minimum length: 1
* Maximum length: 63
* This string is internally validated to prevent XSS, Unicode, directory traversal, and SQL-injection attacks.


|roles
|array[string]
|False
a|Array containing the UUIDs of child roles. Each UUID in the array represents a role that is a child of the aggregate (parent) role.


|tags
|array[link:#tags[tags]]
|False
a|JSON array of tags (name/value pairs).


|type
|string
|True
a|Media type of the resource.
Defined values are:

* "application/vnd.netapp.bxp.role"


|version
|string
|True
a|Version of the resource.
Defined values are:

* "1.0"
