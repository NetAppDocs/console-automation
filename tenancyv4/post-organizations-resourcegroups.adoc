= Add resource groups to an organization

[.api-doc-operation .api-doc-operation-post]#POST# [.api-doc-code-block]#`/organizations/{organization_id}/resourcegroups`#



Adds resource groups to the specified organization.


== Parameters

[cols=5*,options=header]
|===
|Name
|Type
|In
|Required
|Description

|organization_id
|string
|path
|True
a|ID of the organization.

* example: 666a3f38-d4fa-5b62-a391-a69029758d32


|===

== Request Body

Indicates the resource groups to be added.
                   

[cols=4*,options=header]
|===
|Name
|Type
|Required
|Description

|resourcegroups
|array[link:#resourcegroups[resourcegroups]]
|True
a|

|type
|string
|True
a|Media type of the resource.
Defined values are:

* "application/vnd.netapp.bxp.resourcegroupbulk"


|version
|string
|True
a|Version of the resource.
Defined values are:

* "1.0"


|===


.Example request
[%collapsible%closed]
====
[source,json,subs=+macros]
{
  "resourcegroups": [
    {
      "id": "a94ac68f-ea20-47aa-add8-8d41be0d8466",
      "name": "StorageAdmin"
    },
    {
      "name": "StorageAdmin"
    }
  ],
  "type": "application/vnd.netapp.bxp.resourcegroupbulk",
  "version": "1.0"
}
====
