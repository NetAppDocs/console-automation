= Delete a grant

[.api-doc-operation .api-doc-operation-delete]#DELETE# [.api-doc-code-block]#`/partnerships/{partnership_id}/grants/{grant_id}`#



Deletes the specified grant from a partnership.


== Parameters

[cols=5*,options=header]
|===
|Name
|Type
|In
|Required
|Description

|partnership_id
|string
|path
|True
a|ID of the partnership resource. A partnership includes a type and a provider organization.

* example: 666a3f38-d4fa-5b62-a391-a69029758d32


|grant_id
|string
|path
|True
a|Grant ID.

* example: 666a3f38-d4fa-5b62-a391-a69029758d32


|===

== Response

```
Status: 204, Indicates that the grant resource was successfully deleted.
```
