= Associate users in an organization with a role

[.api-doc-operation .api-doc-operation-post]#POST# [.api-doc-code-block]#`/organizations/{organization_id}/roles/{role_id}/users`#



Associates users with a role under the specified organization.


== Parameters

[cols=5*,options=header]
|===
|Name
|Type
|In
|Required
|Description

|organization_id
|string
|path
|True
a|

|role_id
|string
|path
|True
a|

|===

== Request Body

Indicates the desired values to associate multiple users with a specific role under an organization.
                   

[cols=4*,options=header]
|===
|Name
|Type
|Required
|Description

|type
|string
|True
a|Media type of the resource.
Defined values are:

* "application/vnd.netapp.bxp.userbulk"


|users
|array[link:#users[users]]
|False
a|JSON array of users.


|version
|string
|True
a|Version of the resource.
Defined values are:

* "1.0"


|===


.Example request
[%collapsible%closed]
====
[source,json,subs=+macros]
{
  "type": "application/vnd.netapp.bxp.userbulk",
  "users": [
    {
      "aadId": "a94ac68f-ea20-47aa-add8-8d41be0d8466",
      "userId": "e837c373-2ec4-47a8-b016-ba430ebce321"
    },
    {
      "auth0Id": "auth0|5a09551d0d9ef535b823bd3f",
      "userId": "f837c373-2ec4-47a8-b016-ba430ebce321"
    }
  ],
  "version": "1.0"
}
====
