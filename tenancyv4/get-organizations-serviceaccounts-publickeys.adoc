= List all the public keys for a service account

[.api-doc-operation .api-doc-operation-get]#GET# [.api-doc-code-block]#`/organizations/{organization_id}/serviceaccounts/{serviceaccount_id}/publickeys`#



Lists all public keys for the specified service account.


== Parameters

[cols=5*,options=header]
|===
|Name
|Type
|In
|Required
|Description

|organization_id
|string
|path
|True
a|ID of the organization.

* example: 666a3f38-d4fa-5b62-a391-a69029758d32


|serviceaccount_id
|string
|path
|True
a|ID of the service account.

* example: 118df263-44d0-4154-8daa-ff61cb471557


|===

== Response

```
Status: 200, List of all public keys for the service account.
```
