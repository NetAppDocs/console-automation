= Partially update a public key for a service account

[.api-doc-operation .api-doc-operation-patch]#PATCH# [.api-doc-code-block]#`/organizations/{organization_id}/serviceaccounts/{serviceaccount_id}/publickeys/{publickey_id}`#



Updates a public key for the specified service account with partial information.


== Parameters

[cols=5*,options=header]
|===
|Name
|Type
|In
|Required
|Description

|organization_id
|string
|path
|True
a|ID of the organization.

* example: 666a3f38-d4fa-5b62-a391-a69029758d32


|serviceaccount_id
|string
|path
|True
a|ID of the service account.

* example: 118df263-44d0-4154-8daa-ff61cb471557


|publickey_id
|string
|path
|True
a|ID of the public key.

* example: 778f8579-a5b1-48ab-91c5-361acc2ee7fe


|===

== Request Body

Replaces the specified attributes in the request body with the values provided.
                   

[cols=4*,options=header]
|===
|Name
|Type
|Required
|Description

|expirationTimestamp
|string
|False
a|The date and time when the public key expires. This is only applicable when the public key is an RSA key.


|stateDesired
|string
|False
a|Expected state of the public key. Defined values are:

* "ENABLED" - The public key is enabled.
* "DISABLED" - The public key is disabled.


|type
|string
|True
a|Media type of the resource.
Defined values are:

* "application/vnd.netapp.bxp.publickey"


|version
|string
|True
a|Version of the resource.
Defined values are:

* "1.0"


|===


.Example request
[%collapsible%closed]
====
[source,json,subs=+macros]
{
  "stateDesired": "DISABLED",
  "type": "application/vnd.netapp.bxp.publickey",
  "version": "1.0"
}
====

== Response

```
Status: 204, Indicates if the public key was updated.
```
