= Create a public key for a service account

[.api-doc-operation .api-doc-operation-post]#POST# [.api-doc-code-block]#`/organizations/{organization_id}/serviceaccounts/{serviceaccount_id}/publickeys`#



Creates a public key for the specified service account.


== Parameters

[cols=5*,options=header]
|===
|Name
|Type
|In
|Required
|Description

|organization_id
|string
|path
|True
a|ID of the organization.

* example: 666a3f38-d4fa-5b62-a391-a69029758d32


|serviceaccount_id
|string
|path
|True
a|ID of the service account.

* example: 118df263-44d0-4154-8daa-ff61cb471557


|===

== Request Body

Indicates the desired values for the public key to be created.
                   

[cols=4*,options=header]
|===
|Name
|Type
|Required
|Description

|expirationTimestamp
|string
|False
a|The date and time when the public key expires. This is only applicable when an RSA key is provided for  the public key. When a certificate is provided, the value is set to the certificate's expiration timestamp.


|publicKey
|string
|True
a|Public X.509 certificate or RSA key in PEM format converted to a string with escaped new line characters (`\n`).


|type
|string
|True
a|Media type of the resource.
Defined values are:

* "application/vnd.netapp.bxp.publickey"


|version
|string
|True
a|Version of the resource.
Defined values are:

* "1.0"


|===


.Example request
[%collapsible%closed]
====
[source,json,subs=+macros]
{
  "expirationTimestamp": "2026-06-24T16:25:08Z",
  "publicKey": "-----BEGIN PUBLIC KEY-----\\nMIIBIjANBgkqhkiG9w0B...\\n-----END PUBLIC KEY-----\\n",
  "type": "application/vnd.netapp.bxp.publickey",
  "version": "1.0"
}
====

== Response

```
Status: 201, Returns information about the newly created public key in the JSON response body.
```


[cols=4*,options=header]
|===
|Name
|Type
|Required
|Description

|audience
|string
|True
a|The value that must be included in the 'audience' claim of the private key JWT token.


|expirationTimestamp
|string
|True
a|The date and time when the public key expires.


|fingerprint
|string
|True
a|The SHA-1 fingerprint of the public X.509 certificate or RSA key that was provided for the public key.


|id
|string
|True
a|Globally unique identifier of the resource conforming to the UUIDv4 schema.


|keyType
|string
|True
a|Type of the public key.
Defined values are:

* "X509_CERTIFICATE" - The public key is an X.509 certificate.
* "RSA_KEY" - The public key is an RSA key.


|metadata
|link:#type_bxp_metadata[type_bxp_metadata]
|True
a|Metadata associated with the resource.


|state
|string
|True
a|The state of the public key. Defined values are:

* "ENABLED" - The public key is enabled.
* "DISABLED" - The public key is disabled.


|type
|string
|True
a|Media type of the resource.
Defined values are:

* "application/vnd.netapp.bxp.publickey"


|version
|string
|True
a|Version of the resource.
Defined values are:

* "1.0"


|===


.Example response
[%collapsible%closed]
====
[source,json,subs=+macros]
{
  "audience": "https://netapp-cloud-account.auth0.com/",
  "expirationTimestamp": "2026-06-24T16:25:08Z",
  "fingerprint": "33:0E:66:6A:D0:89:EA:78:10:7D:5A:35:FF:C5:51:E9:3A:CB:47:A8",
  "id": "4336388b-2992-43af-81de-ba9284b7dc36",
  "keyType": "RSA_KEY",
  "metadata": {
    "createdBy": "8f84cf09-8036-51e4-b579-bd30cb07b269",
    "creationTimestamp": "2025-06-24T16:25:08Z",
    "modificationTimestamp": "2025-06-24T16:25:08Z",
    "modifiedBy": "8f84cf09-8036-51e4-b579-bd30cb07b269"
  },
  "state": "ENABLED",
  "type": "application/vnd.netapp.bxp.publickey+json",
  "version": "1.0"
}
