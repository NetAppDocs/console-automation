= Update a folder

[.api-doc-operation .api-doc-operation-put]#PUT# [.api-doc-code-block]#`/folders/{folder_id}`#



Replaces the specified folder with new information.


== Parameters

[cols=5*,options=header]
|===
|Name
|Type
|In
|Required
|Description

|folder_id
|string
|path
|True
a|ID of the folder.

* example: 666a3f38-d4fa-5b62-a391-a69029758d32


|===
