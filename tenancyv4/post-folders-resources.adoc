= Add resources to a folder

[.api-doc-operation .api-doc-operation-post]#POST# [.api-doc-code-block]#`/folders/{folder_id}/resources`#



Adds resources to the specified folder.


== Parameters

[cols=5*,options=header]
|===
|Name
|Type
|In
|Required
|Description

|folder_id
|string
|path
|True
a|The ID of the folder that contains the resources.

* example: 666a3f38-d4fa-5b62-a391-a69029758d32


|===

== Request Body

Indicates the resources to add to the folder.
                   

[cols=4*,options=header]
|===
|Name
|Type
|Required
|Description

|resources
|array[string]
|True
a|JSON array of resource UUIDs.


|type
|string
|True
a|Media type of the resource.
Defined values are:

* "application/vnd.netapp.bxp.resourcebulk"


|version
|string
|True
a|Version of the resource.
Defined values are:

* "1.0"


|===


.Example request
[%collapsible%closed]
====
[source,json,subs=+macros]
{
  "resources": [
    "a94ac68f-ea20-47aa-add8-8d41be0d8466",
    "b04ac68f-da27-578a-cee8-9e45bq0d8466"
  ],
  "type": "application/vnd.netapp.bxp.resourcebulk",
  "version": "1.0"
}
====

== Response

```
Status: 202, Indicates the resources request was accepted.
```
