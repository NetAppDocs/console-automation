= Retrieve a user with a grant in a partnership

[.api-doc-operation .api-doc-operation-get]#GET# [.api-doc-code-block]#`/partnerships/{partnership_id}/grants/{grant_id}/users/{user_id}`#



Retrieves a user in the specified grant in a partnership.


== Parameters

[cols=5*,options=header]
|===
|Name
|Type
|In
|Required
|Description

|partnership_id
|string
|path
|True
a|ID of the partnership.

* example: 666a3f38-d4fa-5b62-a391-a69029758d32


|user_id
|string
|path
|True
a|ID of the user.

* example: 118df263-44d0-4154-8daa-ff61cb471557


|grant_id
|string
|path
|True
a|ID of the grant used to display users.

* example: 778f8579-a5b1-48ab-91c5-361acc2ee7fe


|===
