= Permission endpoints in the identity and access management REST API for the NetApp Console



The permission endpoints provide access to an internal collection of available permissions in NetApp Console identity and access management (IAM). Permissions are defined as a low-level string of characters that a user or service account can use to determine authorization.

A user can have permissions for the resources in the entire organization or in specific folders and projects. Permissions prevent or allow a user to perform specific actions at a specific level of the resource hierarchy.
