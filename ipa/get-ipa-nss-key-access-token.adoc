= Retrieve an access token

[.api-doc-operation .api-doc-operation-get]#GET# [.api-doc-code-block]#`/ipa/nss-key/{nssKeyId}/access-token`#



Retrieves a NetApp Support Site user B2C access token.
            +
 +
*Token usage:* NetApp Console service token.


== Parameters

[cols=5*,options=header]
|===
|Name
|Type
|In
|Required
|Description

|nssKeyId
|string
|path
|True
a|

|X-Service-Request-Id
|string
|header
|False
a|Request identifier


|===
