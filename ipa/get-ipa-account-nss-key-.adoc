= Retrieve NetApp Support Site key credentials

[.api-doc-operation .api-doc-operation-get]#GET# [.api-doc-code-block]#`/ipa/account/{accountId}/nss-key/{nssKeyId}`#



Retrieves the NetApp Support Site key credentials for a specified NetApp Support Site ID.
            +
 +
*Token usage:* NetApp Console user token.
