= Create an IDaaS NetApp Support Site key credential

[.api-doc-operation .api-doc-operation-post]#POST# [.api-doc-code-block]#`/ipa/account/{accountId}/idaas-key`#



Creates an identity-as-a-service (IDaaS) NetApp Support Site key credential.
            +
 +
*Token usage:* NetApp Console user token.
