= Retrieve NetApp Support Site credentials

[.api-doc-operation .api-doc-operation-get]#GET# [.api-doc-code-block]#`/ipa/account/{accountId}/nss-key/resource/{resourceId}`#



Retrieves NetApp Support Site credentials for a specified resource ID and validates them if the validateKey field is true.
            +
*Token usage:* NetApp Console user token.
