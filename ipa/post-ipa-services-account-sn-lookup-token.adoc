= Retrieve an IPA serial number token

[.api-doc-operation .api-doc-operation-post]#POST# [.api-doc-code-block]#`/ipa/services/account/{accountId}/sn-lookup-token`#



Retrieves an IPA serial number token.
            +
 +
*Token usage:* NetApp Console service token.
