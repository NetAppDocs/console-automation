= Retrieve a list of NetApp Support Site key credentials

[.api-doc-operation .api-doc-operation-get]#GET# [.api-doc-code-block]#`/ipa/account/{accountId}/nss-keys`#



Retrieves a list of NetApp Support Site key credentials associated with an account. You can optionally choose to include the resource data associated with a NetApp Support Site ID.
                          +
 +
*Token usage:* NetApp Console user token.
