= Retrieve support case details

[.api-doc-operation .api-doc-operation-get]#GET# [.api-doc-code-block]#`/ipa/support/case/{caseNumber}/details`#



Retrieves support case details.
            +
 +
*Token usage:* Both a NetApp Console user token and service token. The service token needs to include one of the following scopes: 'occm-access', 'ipa-internal'.
