= Retrieve support cases

[.api-doc-operation .api-doc-operation-get]#GET# [.api-doc-code-block]#`/ipa/support/case-search`#



Retrieves a list of support cases according to the filter parameters.
            +
 +
*Token usage:* Both a NetApp Console user token and service token. The service token needs to include one of the following scopes: 'occm-access', 'ipa-internal'.
