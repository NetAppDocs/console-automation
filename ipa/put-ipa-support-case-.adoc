= Update support case notes or status

[.api-doc-operation .api-doc-operation-put]#PUT# [.api-doc-code-block]#`/ipa/support/case/{caseNumber}`#



Updates the support case notes or status.
            +
 +
*Token usage:* Both a NetApp Console user token and service token. The service token needs to include one of the following scopes: 'occm-access', 'ipa-internal'.
