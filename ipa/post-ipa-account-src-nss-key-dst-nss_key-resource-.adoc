= Update a resource NetApp Support Site key

[.api-doc-operation .api-doc-operation-post]#POST# [.api-doc-code-block]#`/ipa/account/{accountId}/src-nss-key/{srcNssKeyId}/dst-nss_key/{dstNssKeyId}/resource/{resourceId}`#



Transfers the source NetApp Support Site data to the destination NetApp Support Site key of the associated resource.
            +
*Token usage:* NetApp Console user token.
