= Retrieve on-prem ONTAP data

[.api-doc-operation .api-doc-operation-get]#GET# [.api-doc-code-block]#`/license/onPrem/account/{accountId}/clusters`#



Retrieves on-premises ONTAP cluster data. +
The ONTAP cluster must be configured for AutoSupport to send back the telemetry and configuration data to NetApp.

Data retrieved by this endpoint is comprised mostly of ONTAP cluster configuration details (such as the cluster management IP address, capacity, contract date, feature licenses, and more).

*Token usage:* NetApp Console user token.
