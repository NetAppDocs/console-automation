= Retrieve the preferred subscription ID for an account

[.api-doc-operation .api-doc-operation-get]#GET# [.api-doc-code-block]#`/license/account/{accountId}/subscriptions/preferred`#



Retrieves the preferred subscription ID for a specified account ID.

**Note:**

* This API authorizes user calls and service calls with a predefined scope only.

*Token usage:* Either a NetApp Console user token or service token.
