= Retrieve all subscriptions for an account

[.api-doc-operation .api-doc-operation-get]#GET# [.api-doc-code-block]#`/license/account/{accountId}/subscriptions`#



Retrieves the subscription information for a specified subscription.

**Note:**

* In the context of an account provided in URL parameters, the API returns subscriptions for all accounts.
* If the subscription ID is provided, the API returns only the subscription details or an empty response.
* The API authorizes three types of calls: user calls, agent calls (Console agent), and service calls (with a predefined scope).
* For a service call, you must send the subscription type as a query parameter (the API validates the scope against the subscription type).

*Token usage:* NetApp Console service token.
