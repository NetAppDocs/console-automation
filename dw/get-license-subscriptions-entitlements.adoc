= Retrieve the subscription entitlements

[.api-doc-operation .api-doc-operation-get]#GET# [.api-doc-code-block]#`/license/subscriptions/entitlements`#



Retrieves the subscription entitlements information for a specified service name.

**Note:**

* This API authorizes service calls with a predefined scope only.

{blank} +
*Token usage:* NetApp Console service token with the following scopes: %scopes%.
