= Retrieve a list of CryptoKeys

[.api-doc-operation .api-doc-operation-get]#GET# [.api-doc-code-block]#`/account/{accountId}/providers/cloudmanager_cbs/api/v1/gcp/project/{projectId}/region/{locationId}/keyrings/{keyRingId}/cryptokey`#



Retrieves a list of CryptoKeys for a KeyRing in a region from a specific Google Cloud project.

**Token usage:** NetApp Console user token.
