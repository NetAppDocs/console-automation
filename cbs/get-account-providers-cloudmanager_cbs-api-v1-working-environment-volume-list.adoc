= Retrieve system volume directories

[.api-doc-operation .api-doc-operation-get]#GET# [.api-doc-code-block]#`/account/{accountId}/providers/cloudmanager_cbs/api/v1/working-environment/{workingEnvironmentId}/volume/{volumeId}/list`#



Retrieves a list of system volume directories. You can run a query by using a target storage VM and the path of directories to browse within the volume.

**Token usage:** NetApp Console user token.
