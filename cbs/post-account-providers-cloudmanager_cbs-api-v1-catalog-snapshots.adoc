= Retrieve snapshots

[.api-doc-operation .api-doc-operation-post]#POST# [.api-doc-code-block]#`/account/{accountId}/providers/cloudmanager_cbs/api/v1/catalog/snapshots`#



Retrieves a list of local and cloud snapshots related to a particular volume by using the catalog index query.

**Token usage:** NetApp Console user token.
