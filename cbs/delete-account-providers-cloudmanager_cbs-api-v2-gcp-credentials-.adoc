= Delete Google Cloud credentials

[.api-doc-operation .api-doc-operation-delete]#DELETE# [.api-doc-code-block]#`/account/{accountId}/providers/cloudmanager_cbs/api/v2/gcp/credentials/{credentialsId}`#



Deletes the Google Cloud credentials for a Google Cloud project in the NetApp Console SaaS application.

**Token usage:** NetApp Console user token.
