= Retrieve regions

[.api-doc-operation .api-doc-operation-get]#GET# [.api-doc-code-block]#`/account/{accountId}/providers/cloudmanager_cbs/api/v1/azure/subscription/{subscriptionId}/region`#



Retrieves a list of regions from a specific subscription ID in Azure.

**Token usage:** NetApp Console user token.
