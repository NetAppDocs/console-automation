= Delete all snapshots (volume)

[.api-doc-operation .api-doc-operation-delete]#DELETE# [.api-doc-code-block]#`/account/{accountId}/providers/cloudmanager_cbs/api/v1/backup/working-environment/{workingEnvironmentId}/volume/{volumeId}/snapshot`#



Deletes all the snapshots for a specified volume backed up by using the NetApp Backup and Recovery service for a system.

**Token usage:** NetApp Console user token.


== Parameters

[cols=5*,options=header]
|===

|Name
|Type
|In
|Required
|Description

|force
|boolean
|query
|False
a|If set to true, all the metadata for the snapshots will be cleaned from the database. Please check if the snapshots are in the object store and then clean them.

* Default value:


|remove-object-store-config
|boolean
|query
|False
a|If set to true, the object store configuration that is not being used by any volume will be cleaned up from the system.

* Default value:


|accountId
|string
|path
|True
a|Account ID


|workingEnvironmentId
|string
|path
|True
a|System ID


|volumeId
|string
|path
|True
a|File system ID/Volume ID


|authorization
|string
|header
|True
a|User token


|X-Agent-Id
|string
|header

a|Request identifier

|===
