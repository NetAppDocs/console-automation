= Update Google Cloud credentials

[.api-doc-operation .api-doc-operation-put]#PUT# [.api-doc-code-block]#`/account/{accountId}/providers/cloudmanager_cbs/api/v2/gcp/credentials/{credentialsId}`#



Updates Google Cloud credentials for a Google Cloud project in the NetApp Console SaaS application.

**Token usage:** NetApp Console user token.


== Parameters

[cols=5*,options=header]
|===

|Name
|Type
|In
|Required
|Description

|accountId
|string
|path
|True
a|Account ID


|credentialsId
|string
|path
|True
a|Credential ID


|authorization
|string
|header
|True
a|User token


|X-Agent-Id
|string
|header
|True
a|Request identifier

|===

== Request Body

                

[cols=4*,options=header]
|===
|Name
|Type
|Required
|Description

|access-key
|string
|False
a|Google Cloud access-key.


|secret-password
|string
|False
a|Google Cloud secret-password.


|===


.Example request
[%collapsible%closed]
====
[source,json,subs=+macros]
{
  "access-key": "string",
  "secret-password": "string"
}
====
