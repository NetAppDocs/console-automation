= Retrieve Azure virtual networks

[.api-doc-operation .api-doc-operation-get]#GET# [.api-doc-code-block]#`/account/{accountId}/providers/cloudmanager_cbs/api/v1/azure/subscription/{subscriptionId}/vnet`#



Retrieves a list of Azure virtual networks from a specific subscription ID in Azure cloud.

**Token usage:** NetApp Console user token.
