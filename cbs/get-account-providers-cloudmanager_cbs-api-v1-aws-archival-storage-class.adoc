= Retrieve AWS storage classes

[.api-doc-operation .api-doc-operation-get]#GET# [.api-doc-code-block]#`/account/{accountId}/providers/cloudmanager_cbs/api/v1/aws/archival-storage-class`#



Retrieves a list of AWS storage classes available for archival support.

**Token usage:** NetApp Console user token.
