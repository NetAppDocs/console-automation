= Remove backup settings

[.api-doc-operation .api-doc-operation-delete]#DELETE# [.api-doc-code-block]#`/account/{accountId}/providers/cloudmanager_cbs/api/v1/backup/working-environment/{workingEnvironmentId}/volume/{volumeId}`#



Removes the backup settings on a specified backed up volume. To disable backup on a volume, the volume snapshots must be deleted.

**Token usage:**NetApp Console user token.
