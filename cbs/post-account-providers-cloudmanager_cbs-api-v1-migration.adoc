= Trigger backup migration

[.api-doc-operation .api-doc-operation-post]#POST# [.api-doc-code-block]#`/account/{accountId}/providers/cloudmanager_cbs/api/v1/migration`#



Triggers the migration of backups from the source ONTAP cluster to the target ONTAP cluster.


== Parameters

[cols=5*,options=header]
|===

|Name
|Type
|In
|Required
|Description

|accountId
|string
|path
|True
a|Account ID


|authorization
|string
|header
|True
a|
|===
