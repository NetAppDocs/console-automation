= Retrieve Google Cloud projects

[.api-doc-operation .api-doc-operation-get]#GET# [.api-doc-code-block]#`/account/{accountId}/providers/cloudmanager_cbs/api/v1/gcp/project`#



Retrieves a list of Google Cloud projects.

**Token usage:** NetApp Console user token.
