= Cloud endpoints in the NetApp Backup and Recovery REST API



The cloud endpoints enable you to retrieve information about various resources of the AWS, Azure, Google Cloud providers from the NetApp Console SaaS application.

You can perform the following actions by using the cloud endpoints:

* Retrieve the accounts, AWS region keys, AWS region private links, AWS archival storage class, Azure subscription, Azure subscription regions, Azure subscription resource-group, Azure subscription VNet, Azure subscription vault key, Google Cloud projects, Google Cloud project region, Google KeyRings and more. +

* Create a subscription resource-group. +
