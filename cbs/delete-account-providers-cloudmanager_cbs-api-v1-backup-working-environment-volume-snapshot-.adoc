= Delete a snapshot

[.api-doc-operation .api-doc-operation-delete]#DELETE# [.api-doc-code-block]#`/account/{accountId}/providers/cloudmanager_cbs/api/v1/backup/working-environment/{workingEnvironmentId}/volume/{volumeId}/snapshot/{snapshotId}`#



Deletes a specified snapshot of a specified volume backed up using the NetApp Backup and Recovery service for a system.

**Token usage:** NetApp Console user token.
