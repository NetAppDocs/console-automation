= Modify a replication policy in a system

[.api-doc-operation .api-doc-operation-patch]#PATCH# [.api-doc-code-block]#`/account/{accountId}/providers/cloudmanager_cbs/api/v1/working-environment/{workingEnvironmentId}/replication-policy/{policyId}`#



Modifies a replication policy in a system.

**Token usage:** NetApp Console user token.


== Parameters

[cols=5*,options=header]
|===

|Name
|Type
|In
|Required
|Description

|accountId
|string
|path
|True
a|Account ID


|workingEnvironmentId
|string
|path
|True
a|System ID


|policyId
|string
|path
|True
a|Replication policy UUID


|authorization
|string
|header
|True
a|User token


|X-Agent-Id
|string
|header
|True
a|Request identifier

|===

== Request Body

                

[cols=4*,options=header]
|===
|Name
|Type
|Required
|Description

|rule
|array[link:#rule[rule]]
|False
a|

|transfer-schedule
|string
|False
a|Cron-based schedule for transfers. To disable the schedule, pass the value as: none(case sensitive). By default, an hourly schedule is selected.


|===


.Example request
[%collapsible%closed]
====
[source,json,subs=+macros]
{
  "rule": [
    {
      "label": "string",
      "retention": "string"
    }
  ],
  "transfer-schedule": "string"
}
====
