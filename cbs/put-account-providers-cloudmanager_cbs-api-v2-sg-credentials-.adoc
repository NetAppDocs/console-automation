= Update StorageGRID credentials

[.api-doc-operation .api-doc-operation-put]#PUT# [.api-doc-code-block]#`/account/{accountId}/providers/cloudmanager_cbs/api/v2/sg/credentials/{storageServerName}`#



Updates StorageGRID credentials in the NetApp Console SaaS application.

**Token usage:** NetApp Console user token.


== Parameters

[cols=5*,options=header]
|===

|Name
|Type
|In
|Required
|Description

|accountId
|string
|path
|True
a|Account ID


|storageServerName
|string
|path
|True
a|StorageGRID server name


|authorization
|string
|header
|True
a|User token


|X-Agent-Id
|string
|header
|True
a|Request identifier

|===

== Request Body

                

[cols=4*,options=header]
|===
|Name
|Type
|Required
|Description

|port
|string
|False
a|StorageGRID node port.


|access-key
|string
|False
a|StorageGRID tenant access-key.


|secret-password
|string
|False
a|StorageGRID tenant secret-password.


|===


.Example request
[%collapsible%closed]
====
[source,json,subs=+macros]
{
  "port": "string",
  "access-key": "string",
  "secret-password": "string"
}
