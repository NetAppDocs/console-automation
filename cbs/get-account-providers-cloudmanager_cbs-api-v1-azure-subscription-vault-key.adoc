= Retrieve Azure vault keys

[.api-doc-operation .api-doc-operation-get]#GET# [.api-doc-code-block]#`/account/{accountId}/providers/cloudmanager_cbs/api/v1/azure/subscription/{subscriptionId}/vault/{vaultName}/key`#



Retrieves a list of Azure vault keys from a particular vault for a specific subscription ID in Azure cloud.

**Token usage:** NetApp Console user token.
