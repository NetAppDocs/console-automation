= Retrieve AWS private links

[.api-doc-operation .api-doc-operation-get]#GET# [.api-doc-code-block]#`/account/{accountId}/providers/cloudmanager_cbs/api/v1/aws/awsAccountId/{awsAccountId}/region/{regionCode}/privateLink`#



Retrieves a list of AWS private links from a specified region that belongs to a specified AWS account ID.

**Token usage:** NetApp Console user token.
