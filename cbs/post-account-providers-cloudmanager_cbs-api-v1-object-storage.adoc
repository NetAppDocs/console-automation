= Create a bucket for a cloud provider

[.api-doc-operation .api-doc-operation-post]#POST# [.api-doc-code-block]#`/account/{accountId}/providers/cloudmanager_cbs/api/v1/object-storage`#



Creates a bucket for the specified cloud provider.


== Parameters

[cols=5*,options=header]
|===

|Name
|Type
|In
|Required
|Description

|account-id
|string
|query
|False
a|Account ID for the AWS provider


|project-id
|string
|query
|False
a|Project ID for the Google Cloud provider


|resource-group
|string
|query
|False
a|Resource group for the Azure provider


|subscription
|string
|query
|False
a|Subscription for the Azure provider


|credential-id
|string
|query
|False
a|Credential ID for the SGWS or ONTAP S3 provider


|accountId
|string
|path
|True
a|Account ID


|job-id
|string
|header
|True
a|Job ID for the create bucket operation


|authorization
|string
|header
|True
a|Authorization token


|X-Agent-Id
|string
|header
|True
a|Agent ID

|===
