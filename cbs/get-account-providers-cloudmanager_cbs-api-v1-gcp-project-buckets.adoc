= Retrieve a list of Google Cloud buckets in a region

[.api-doc-operation .api-doc-operation-get]#GET# [.api-doc-code-block]#`/account/{accountId}/providers/cloudmanager_cbs/api/v1/gcp/project/{projectId}/buckets`#



Retrieves a list of buckets in a region from a specific Google Cloud project from the NetApp Console SaaS application.

**Token usage:** NetApp Console user token.
