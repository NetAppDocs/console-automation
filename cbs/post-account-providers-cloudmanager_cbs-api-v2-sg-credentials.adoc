= Register new StorageGRID credentials

[.api-doc-operation .api-doc-operation-post]#POST# [.api-doc-code-block]#`/account/{accountId}/providers/cloudmanager_cbs/api/v2/sg/credentials`#



Registers new StorageGRID credentials in the NetApp Console SaaS application.

**Token usage:** NetApp Console user token.


== Parameters

[cols=5*,options=header]
|===

|Name
|Type
|In
|Required
|Description

|accountId
|string
|path
|True
a|Account ID


|authorization
|string
|header
|True
a|User token


|X-Agent-Id
|string
|header
|True
a|Request identifier

|===

== Request Body

                

[cols=4*,options=header]
|===
|Name
|Type
|Required
|Description

|storage-server-name
|string
|True
a|StorageGRID server name.


|port
|string
|True
a|StorageGRID node port.


|access-key
|string
|True
a|StorageGRID tenant access-key.


|secret-password
|string
|True
a|StorageGRID tenant secret-password.


|===


.Example request
[%collapsible%closed]
====
[source,json,subs=+macros]
{
  "storage-server-name": "string",
  "port": "string",
  "access-key": "string",
  "secret-password": "string"
}
====

== Response

```
Status: 200, Default response
```
