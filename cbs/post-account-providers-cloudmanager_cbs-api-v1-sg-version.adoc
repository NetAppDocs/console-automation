= Retrieve StorageGRID server version

[.api-doc-operation .api-doc-operation-post]#POST# [.api-doc-code-block]#`/account/{accountId}/providers/cloudmanager_cbs/api/v1/sg/version`#



Retrieves the version of the StorageGRID servers.

**Token usage:** NetApp Console user token.
