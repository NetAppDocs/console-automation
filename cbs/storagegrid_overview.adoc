= StorageGRID endpoints in the NetApp Backup and Recovery REST API



The StorageGRID API endpoints retrieve version and other details for the StorageGRID server. You can also discover StorageGRID WebScale server in the NetApp Console SaaS application.

You can perform the following operations by using the StorageGRID endpoints: +

* Retrieve StorageGRID server version
* Retrieve details of the StorageGRID server.
* Discover or add new StorageGRID server into the NetApp Console. +
