= Create a replication policy in a system

[.api-doc-operation .api-doc-operation-post]#POST# [.api-doc-code-block]#`/account/{accountId}/providers/cloudmanager_cbs/api/v1/working-environment/{workingEnvironmentId}/replication-policy`#



Creates a replication policy in a system.

**Token usage:** NetApp Console user token.


== Parameters

[cols=5*,options=header]
|===

|Name
|Type
|In
|Required
|Description

|accountId
|string
|path
|True
a|Account ID


|workingEnvironmentId
|string
|path
|True
a|System ID


|authorization
|string
|header
|True
a|User token


|X-Agent-Id
|string
|header
|True
a|Request identifier

|===

== Request Body

                

[cols=4*,options=header]
|===
|Name
|Type
|Required
|Description

|name
|string
|True
a|

|rule
|array[link:#rule[rule]]
|False
a|

|transfer-schedule
|string
|False
a|Cron-based schedule for transfers. To disable the schedule, pass the value as: none(case sensitive). By default, an hourly schedule is selected.


|copy-all-source-snapshots
|boolean
|False
a|Copies all source snapshots to the replication detstination


|copy-latest-source-snapshot
|boolean
|False
a|Copies only the latest source snapshot to the replication detstination


|svm-name
|string
|False
a|Name of the SVM where the replication policy should be created


|===


.Example request
[%collapsible%closed]
====
[source,json,subs=+macros]
{
  "name": "string",
  "rule": [
    {
      "label": "string",
      "retention": "string"
    }
  ],
  "transfer-schedule": "string",
  "svm-name": "string"
}
====

== Response

```
Status: 200, Default response
```
