= Discover a new StorageGRID WebScale server

[.api-doc-operation .api-doc-operation-post]#POST# [.api-doc-code-block]#`/account/{accountId}/providers/cloudmanager_cbs/api/v1/sg`#



Discovers a new StorageGRID server in the NetApp Console SaaS application.

**Token usage:** NetApp Console user token.


== Parameters

[cols=5*,options=header]
|===

|Name
|Type
|In
|Required
|Description

|accountId
|string
|path
|True
a|Account ID


|authorization
|string
|header
|True
a|User token


|X-Agent-Id
|string
|header
|True
a|Request identifier

|===

== Request Body

                

[cols=4*,options=header]
|===
|Name
|Type
|Required
|Description

|admin-host
|string
|True
a|StorageGRID Admin host IP address.


|admin-username
|string
|True
a|StorageGRID Admin username.


|admin-password
|string
|True
a|StorageGRID Admin password.


|===


.Example request
[%collapsible%closed]
====
[source,json,subs=+macros]
{
  "admin-host": "string",
  "admin-username": "string",
  "admin-password": "string"
}
====
