= Retrieve a list of schedules in a system

[.api-doc-operation .api-doc-operation-get]#GET# [.api-doc-code-block]#`/account/{accountId}/providers/cloudmanager_cbs/api/v1/working-environment/{workingEnvironmentId}/schedules`#



Retrieves a list of schedules present in a system.

**Token usage:** NetApp Console user token.


== Parameters

[cols=5*,options=header]
|===

|Name
|Type
|In
|Required
|Description

|accountId
|string
|path
|True
a|

|workingEnvironmentId
|string
|path
|True
a|

|authorization
|string
|header
|True
a|

|X-Agent-Id
|string
|header
|True
a|
|===
