= Retrieve a list of StorageGRID buckets

[.api-doc-operation .api-doc-operation-post]#POST# [.api-doc-code-block]#`/account/{accountId}/providers/cloudmanager_cbs/api/v1/sg/buckets`#



Retrieves a list of StorageGRID buckets from the StorageGRID server.

**Token usage:** NetApp Console user token.

|Name

|True
a|Account ID

|string

|True
a|User token


|X-Agent-Id
|string
|header
|True
a|Request identifier

|===
