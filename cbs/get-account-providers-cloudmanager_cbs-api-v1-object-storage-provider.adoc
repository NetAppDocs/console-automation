= Retrieve a bucket for a cloud provider

[.api-doc-operation .api-doc-operation-get]#GET# [.api-doc-code-block]#`/account/{accountId}/providers/cloudmanager_cbs/api/v1/object-storage/{provider}/{objectStoreName}`#



Retrieves a specific bucket for the specified cloud provider.


== Parameters

[cols=5*,options=header]
|===

|Name
|Type
|In
|Required
|Description

|accountId
|string
|path
|True
a|Account ID


|provider
|string
|path
|True
a|Cloud provider type

* enum: ["AWS", "GCP", "AZURE", "SGWS", "ONTAP_S3"]


|objectStoreName
|string
|path
|True
a|Bucket name


|authorization
|string
|header
|True
a|Authorization token


|X-Agent-Id
|string
|header
|True
a|Agent ID

|===

== Response

```
Status: 200, Default response
```


[cols=4*,options=header]
|===
|Name
|Type
|Required
|Description

|object-store-name
|string
|True
a|Bucket name


|region
|string
|False
a|Bucket region


|provider
|string
|True
a|Cloud provider


|account-id
|string
|False
a|Account ID for the AWS provider


|project-id
|string
|False
a|Project ID for the Google Cloud provider


|subscription
|string
|False
a|Subscription for the Azure provider


|encryption
|string
|False
a|Encryption type used for the bucket


|is-public
|boolean
|False
a|Indicates whether the bucket is public


|object-lock
|string
|False
a|Object lock enabled for the bucket


|resource-group
|string
|False
a|Resource group for the Azure provider


|tags
|array[link:#tags[tags]]
|False
a|

|===
