= Retrieve a list of AWS S3 buckets from a region

[.api-doc-operation .api-doc-operation-get]#GET# [.api-doc-code-block]#`/account/{accountId}/providers/cloudmanager_cbs/api/v1/aws/awsAccountId/{awsAccountId}/buckets`#



Retrieves a list of AWS S3 buckets from a specified region that belongs to a specified AWS account ID from the NetApp Console SaaS application.

**Token usage:** NetApp Console user token.
