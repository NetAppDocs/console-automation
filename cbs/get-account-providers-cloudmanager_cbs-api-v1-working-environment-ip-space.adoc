= Retrieve IP spaces

[.api-doc-operation .api-doc-operation-get]#GET# [.api-doc-code-block]#`/account/{accountId}/providers/cloudmanager_cbs/api/v1/working-environment/{workingEnvironmentId}/ip-space`#



Retrieves a list of IP spaces within a specified system.

**Token usage:** NetApp Console user token.
